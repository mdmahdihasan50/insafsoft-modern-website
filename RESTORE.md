# Restore the complete project

1. Clone the repository.
2. Open `index.html` directly for local preview, or serve the directory with any static web server.
3. Upload the repository contents to `public_html` or another static hosting document root.

All required HTML, CSS, JavaScript, images and documentation are committed. No dependency installation or build step is required.

The ignored `.claude` metadata and archive/log files are not application source and are not needed to restore the website.
