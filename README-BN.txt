InsafSoft Modern Website

ফাইলসমূহ:
- index.html
- styles.css
- script.js

চালানোর নিয়ম:
1) `index.html` ফাইলে ডাবল ক্লিক করুন।
2) ব্রাউজারে ওয়েবসাইট ওপেন হবে।

এডিট করতে:
- লেখা পরিবর্তন: `index.html`
- ডিজাইন পরিবর্তন: `styles.css`
- ডাইনামিক কাজ পরিবর্তন: `script.js`

নোট:
- কন্টাক্ট ফর্মে সাবমিট করলে ডিফল্ট মেইল অ্যাপ ওপেন হবে।
- লাইভ সার্ভারে দিলেই এই ফাইলগুলো চলবে।
